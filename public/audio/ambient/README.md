Drop royalty-free looping ambient files here. The game loads local MP3 or OGG files only and skips missing files without generating fallback tones.

This folder now includes a starter ambience pack for the current worlds and planned future worlds. Each file is still replaceable: keep the same filename and the app will pick up the new local audio automatically.

Expected filenames:

- rain.mp3
- cafe-room.mp3
- bus-ride.mp3
- classroom.mp3
- theater-room.mp3
- restaurant-room.mp3
- airport.mp3
- park.mp3
- beach.mp3
- ocean.mp3
- fireplace.mp3
- wind.mp3
- birds.mp3
- library.mp3
- hospital.mp3

OGG alternatives are also supported with the same base names, for example:

- rain.ogg
- cafe-room.ogg

Source details for the bundled starter pack are recorded in SOURCES.json.
