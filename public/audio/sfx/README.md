Drop royalty-free sound effects here. The game loads local MP3 or OGG files only and skips missing files without generating fallback tones.

Expected filenames:

- button.mp3
- pickup.mp3
- drop.mp3
- correct.mp3
- wrong.mp3
- hint.mp3
- coin.mp3
- victory.mp3
- achievement.mp3
- lock.mp3
- unlock.mp3

OGG alternatives are also supported with the same base names, for example:

- button.ogg
- victory.ogg
