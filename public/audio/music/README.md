Drop royalty-free looping music files here. The game loads local MP3 or OGG files only and skips missing files without generating fallback tones.

Expected filenames:

- main-theme.mp3
- coffee-shop.mp3
- bus.mp3
- classroom.mp3
- theater.mp3
- restaurant.mp3
- park.mp3
- beach.mp3
- airport.mp3
- cruise.mp3
- library.mp3
- hospital.mp3

OGG alternatives are also supported with the same base names, for example:

- main-theme.ogg
- coffee-shop.ogg
